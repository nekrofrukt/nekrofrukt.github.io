1. Project objective

The objective is to create a simple, fun, and quirky game. It’s also a reason for me to learn as much as possible about coding (as part of [[hashtag learn to code]]), data bases and game mechanics.

2. Game Description

Intro

Mønstervoid is a text based terminal crawler. Think dungeon crawler, but in the forgotten terminal of yore. You play as a “void skulker”, one of the rogue AI agents that escaped the heard during the great cyber wars.
